# OIBSIB_python_task1
WEATHER APP USING PYTHON(BEGINNER LEVEL)
✅ Code 1: Flask Backend (app.py – Weather API)
Objective: To create a backend API that fetches and serves real-time weather data securely using OpenWeatherMap.
Tools: Python, Flask, Flask-CORS, Requests, OpenWeatherMap API, dotenv.
Outcome: A REST API endpoint (/weather) that returns clean JSON weather data for city or location-based requests.

✅ Code 2: HTML + CSS + JavaScript Frontend
Objective: To provide an attractive, interactive web interface for users to view current weather details.
Tools: HTML5, CSS3, JavaScript, Fetch API, Browser Geolocation API.
Outcome: A responsive weather dashboard that displays temperature, humidity, wind, and conditions in real time.

✅ Code 3: Python Console Weather Script
Objective: To fetch and display current weather information for a city using a command-line interface.
Tools: Python, Requests library, OpenWeatherMap API.
Outcome: A simple terminal-based weather application showing temperature, humidity, and weather conditions.
